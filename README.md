libsystemd-qt
=============

Qt-only wrapper for Systemd API

##Build and Installation##
It requires cmake >= 2.8.8 and Qt 5.x

    $ mkdir build
    $ cd build
    $ cmake ../ -DCMAKE_INSTALL_PREFIX=/usr
    $ make
    # make install

##Packages
[PKGBUILD](https://aur.archlinux.org/packages/libsystemd-qt-git/) for Arch Linux
